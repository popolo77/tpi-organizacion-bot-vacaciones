# Subir el proyecto a GitHub

1. Crear un repositorio vacío en GitHub, por ejemplo: `tpi-organizacion-bot-vacaciones`.
2. Abrir Git Bash en esta carpeta.
3. Ejecutar, reemplazando USUARIO y REPO:

```bash
git remote add origin https://github.com/USUARIO/REPO.git
git branch -M main
git push -u origin main
```

4. Si GitHub pide contraseña, usar el Personal Access Token como contraseña. No pegar el PAT en README, capturas ni código.
5. Copiar el link del repositorio y pegarlo en el informe antes de exportar a PDF.
