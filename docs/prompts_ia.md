# Evidencia de uso de IA

Durante el desarrollo del Trabajo Integrador se utilizaron herramientas de IA como apoyo para ordenar el alcance, revisar criterios de formato, explicar BPMN y controlar la coherencia entre el proceso modelado y el código.

Consultas representativas:

1. Revisión de formato del documento: espaciado, tipografía y requisitos formales del PDF.
2. Definición del proceso conveniente para el TPI: gestión de vacaciones por ser un flujo administrativo con decisiones, validaciones, estados, datos, aprobaciones y errores.
3. Explicación conceptual de BPMN y relación con el proceso de vacaciones.
4. Revisión de agregados pendientes según rúbrica: manual de usuario y capturas de uso de IA.

Las capturas completas se encuentran en la carpeta `docs/evidencias_ia/` y dos de ellas fueron incorporadas al anexo del informe final.

La IA se utilizó como asistencia y no como sustitución de la autoría. El contenido fue revisado, adaptado y conectado con los apuntes de la materia Organización Empresarial.
